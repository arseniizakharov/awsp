# Code of Conduct

This project expects respectful, practical collaboration.

## Expected Behavior

- Be direct without being hostile.
- Focus disagreement on the design, code, or tradeoff.
- Respect maintainers' time by keeping issues reproducible and scoped.
- Redact security-sensitive AWS details before posting logs or examples.

## Unacceptable Behavior

- Harassment, threats, or targeted personal attacks.
- Sharing another person's private information.
- Posting credentials, tokens, or sensitive AWS account details.
- Repeated off-topic or bad-faith participation after being asked to stop.

## Enforcement

Maintainers may edit or remove comments, lock discussions, or block participants when needed to keep the project usable and safe.
